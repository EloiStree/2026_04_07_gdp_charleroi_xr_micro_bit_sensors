
# Charleroi XR Micro:bit Sensors

Entraînez-vous à contribuer sur GitHub en ajoutant vos fichiers GLB à ce projet.

## Règles

* GLB uniquement
* `Contribution/NOM_PRENOM/ReadMe.md`
  * Nom/Prénom + GitHub + autres informations
* `Contribution/NOM_PRENOM/GLB/your_glb_name/ReadMe.md`
  * Description de l’objet et informations à son sujet
* `Contribution/NOM_PRENOM/GLB/your_glb_name/your_glb_name.glb`

Example: [ici](https://github.com/EloiStree/2026_04_07_gdp_charleroi_xr_micro_bit_sensors/tree/main/Contribution/STREE_ELOI/GLB/DigitSeg16)

### Convention de nommage

* Alphanumérique
* Pas d’espace
* snake_case
* NOM PRENOM en majuscules sans accent
